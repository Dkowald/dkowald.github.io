Sunday, 13 August 2017

URL helpers for absolute url's

A set of extensions to simplify creating absolute URL's in Razor,

The MVC IUrlHelper is often used in Razor to create links to other Action / Routes in an MVC application. 

    @*Get the url for MyController.Index*@
    @Url.Action("Demo", "MyController")
    @*Returns /MyController/Demo*@

The help creates site relative URL paths. The sample above produces /MyController/Demo If you want an actual absolute URL; the easiest way is to provide the protocol as part of the call

Sometimes, you may really want the absolute URL. As per Stack Overflow and Captain Codeman 

    @Url.Action("Demo", "MyController", null, Context.Request.Scheme, null)

Gives me a full URL such as http://localhost:1212/MyController/Demo

To make this a little easier, here's a gist with overloads for Url.ActionAbsolute and Url.RouteUrlAbsolute To use these, just add Absolute to the call 

    @using kwd.gist

    @Url.ActionAbsolute("Demo", "MyController")
